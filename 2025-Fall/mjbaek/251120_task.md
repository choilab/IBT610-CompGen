# 이번 주 과제: Phylogroup 만들기

## 분석 과정

1. **Matrix 파일 생성**
   - 지난 주는 **MMseqs2**를 이용하여 유전자 매트릭스 파일(`matrix`)만 생성.
   - 하지만 MMseqs2는 결과물이 **매트릭스 파일 한 종류**로 제한됨.

2. **Re-analysis with Roary**
   - **Roary**를 사용하여 재분석 수행.
   - Roary 실행 시 core gene, accessory gene 뿐만 아니라 추후 다른 분석이나 시각화에 쓸 수 있는 다양한 형태의 파일을 얻을 수 있음: `gene_presence_absence.csv`, `core_gene_alignment.aln`...

## 결과
: ### Gene Category Summary

| Category         | Strain Coverage (%)       | Number of Genes |
|-----------------|-------------------------|----------------|
| Core genes       | 99% ≤ strains ≤ 100%    | 1044           |
| Soft core genes  | 95% ≤ strains < 99%     | 132            |
| Shell genes      | 15% ≤ strains < 95%     | 1541           |
| Cloud genes      | 0% ≤ strains < 15%      | 5128           |
| **Total genes**  | 0% ≤ strains ≤ 100%     | 7845           |


    
3. 'accessory_binary_genes.newick` 파일을 기반으로 **phylogenetic tree 그림 생성**
  - Tree를 통해 accessory gene 분포에 따른 그룹화 확인 가능
  - 'accessory_binary_genes.newick` 파일은 Soft core + Shell + Cloud genes 를 담고 있음.
