# Roary + MAFFT Pan-genome Analysis Results

## 분석 개요

- **분석 대상:** 214 strains of *Lactiplantibacillus plantarum*
- **Workflow:** Prokka 어노테이션 → Roary pan-genome 분석 → MAFFT core gene alignment

---

## Pan-genome 분류 결과

| 유전자 그룹          | 기준 (%)                 | 유전자군 개수 |
|---------------------|-------------------------|------------|
| **Core genes**      | 99.9% <= strains <= 100 | 1,173      |
| **Soft core genes** | 95% <= strains < 99.9   | 616        |
| **Shell genes**     | 15% <= strains < 95     | 1,907      |
| **Cloud genes**     | 0% <= strains < 15      | 17,274     |
| **Total genes**     | 0% <= strains <= 100    | 20,970     |

---

## 주요 출력 파일

- **core_gene_alignment.aln**: 모든 core gene concat MSA(MAFFT 기반)  
- **gene_presence_absence.csv**: strain × gene 매트릭스, 각 유전자군의 존재/부재 정보  
- **summary.tabular**: 유전자군별 빈도·분포(위 표 내용 원본)  
- 기타: Clustered proteins, accessory/shell/cloud gene 목록 등

---

## 해석 및 응용

- **Core genome**: 1,173개 유전자가 모든 strain에서 발견됨 (종 특성 규명, 진화·기능 분석용)
- **Accessory genome**:
    - Soft core + shell = 2,523개(집단 내 다양성·환경 적응성)
- **Cloud genome**: 17,274개 유전자가 일부 또는 소수 strain에서만 발견됨 (희귀성, 기능적 다양성, 신기능 탐색)
- **총 유전자군**: 20,970
- 데이터는 계통수 분석, surfaceome/기능 유전자 탐색, phylogroup 비교 등에 활용 가능
